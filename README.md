### Quattro

https://github.com/M-18-team1/tetris

- 大学の授業でチーム制作した、テトリスに似た PC ブラウザゲーム
- Vue.js で実装
- 通常のテトリスとは異なり、コストを消費して「スキル」を使用することが出来る

  - 使用出来るスキルは、ゲーム開始前に選択する「職業」によって決まる。例えば「戦士」を選ぶと攻撃的なスキル構成になる一方、「僧侶」の場合は生存重視のスキル構成となる
